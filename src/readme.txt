This directory and its subdirectoires contain the source code for the OmegaEngine and sample applications.

The "build.cmd" script will compile the code using the Visual Studio 2010/2012/2013 build environment on Windows.
Use "OmegaEngine.sln" to open the solution in Visual Studio 2010/2012/2013.
Use FrameOfReference\Game or FrameOfReference\Editor as the startup project.
